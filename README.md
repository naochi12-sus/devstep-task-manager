# Our Goal - タスク管理アプリケーション

チームや個人の進捗をシンプルに記録・管理できるタスク管理アプリケーションです。
操作は至ってシンプル。タスクを追加・消込・編集・削除することが可能です。

## プロジェクト概要

このアプリは、Next.js と Supabase を使用して作成した、ユーザー認証機能付きのタスク管理アプリです。
TOP画面、ログイン画面、新規登録画面、タスク画面（認証後のページリダイレクト機能）を実装しています。

## 機能一覧

- **ユーザー認証機能**
    - 新規ユーザー登録 / ログイン / ログアウト
    - ログインしたユーザー本人のタスクのみが表示されるセキュアな設計（Row Level Security対応）

- **タスク管理機能 (CRUD)**
    - **作成 (Create):** 新しいタスクや日報をテキスト入力で追加できます。
    - **読み取り (Read):** 登録したタスクが一覧で表示されます。タスクが0件の時は専用の空っぽ画面を表示します。
    - **更新 (Update):** チェックアイコンをクリックして「完了 / 未完了」のステータスを簡単に切り替えられます。また、ペンマークのアイコンをクリックすると、その場でタスクの名前を直接編集して保存できます。
    - **削除 (Delete):** ゴミ箱アイコンをクリックしてタスクを完全に削除できます。

## アプリの使い方

1. **アプリを開く:** TOP画面から「新規登録」または「ログイン」へ進みます。
2. **ログイン** メールアドレスとパスワードを入力して、自分専用のアカウントに入ります。未登録の場合は「新規登録」へ進みます。
3. **アカウント作成** メールアドレスとパスワードを入力して、自分専用のアカウントを登録します。登録後、「ログイン」へ進みます。
4. **タスクの管理:** タスク一覧画面にて、タスクの追加・完了・文字の編集・削除を行います。

## 画面一覧

| 画面名             | URL       | 役割・備考                               |
| :----------------- | :-------- | :--------------------------------------- |
| **TOP画面**        | `/`       | アプリの顔となるランディングページです。 |
| **ログイン画面**   | `/login`  | 未ログイン時の自動遷移先です。           |
| **新規登録画面**   | `/signup` | 新しいユーザーを作成できます。           |
| **タスク一覧画面** | `/tasks`  | ログイン後のみアクセス可能です。         |

## 使用技術

- **フロントエンド**: Next.js (App Router), TypeScript, React
- **バックエンド/データベース**: Supabase (PostgreSQL, Authentication)
- **スタイリング**: Tailwind CSS
- **アイコン**: Lucide React
- **デプロイ**: Vercel

## セットアップ手順

1. このリポジトリを自分の PC に保存（クローン）します。
2. ターミナルで `npm install` を実行し、必要な部品（パッケージ）をインストールします。
3. プロジェクトのルートに `.env.local` ファイルを作成し、Supabase の接続情報を設定します。
4. `npm run dev` コマンドで開発用サーバーを起動します。
5. ブラウザで `http://localhost:3000` にアクセスします。

## 環境変数の設定

ローカルの `.env.local` や Vercel の設定画面（Environment Variables）に以下の項目を設定してください。

- `NEXT_PUBLIC_SUPABASE_URL`: Supabase のプロジェクトURL
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`: Supabase の API キー
